Everything to know about HTML

Introduction

HTML stands for Hyper Text Markup Language. It acts as the skeletal structure of a website. It inludes images, texts, links. This is essesntial if one wants to publish any information through Web

Lists

List is a Special feature in HTML used for displaying a list of things.

Types of Lists

  1. Ordered List
  2. Unordered List
  3. Definition List

Ordered List

Ordered lists display items with numbering like the "type of lists" list above. We can use attributes (like 1,2,3,...), (i, ii, iii,...), (I, II, III,...), (A, B, C,...), (a, b, c,...).

Examples
  1. Apple
  2. Banana
  3. Orange
  1. India
  2. China
  3. Japan

Unordered List

Unordered lists display items with bullets or symbols. It is also called "Bulleted Lists". These symbols are customizable too.

Examples
  • India
  • China
  • Japan
  • India
  • China
  • Japan

Definition List

This List is different from the other two lists. This list is not based on list items, but on term definition. Hence does not number or symbolize the list.
